## 2. Checklist SEO para Paginas de Categoria

### Elementos On-Page (OBRIGATORIO em cada pagina de categoria)

- [ ] **Title tag unico** (60 caracteres max): "[Categoria] - Comprar [Categoria] Online | VoltStock"
  - Exemplo: "Placas Graficas - Comprar GPU Online | VoltStock"
- [ ] **Meta description unica** (155 caracteres max) com CTA
- [ ] **H1 unico** por pagina com keyword principal
- [ ] **Texto introdutorio** de 150-300 palavras acima dos produtos (contexto para Google)
- [ ] **Breadcrumbs** com schema markup (Home > Categoria > Subcategoria)
- [ ] **URLs limpas e descritivas**: voltstock.pt/placas-graficas (nao /cat-123)
- [ ] **Imagens optimizadas** com alt text descritivo
- [ ] **Filtros sem duplicar URLs** (usar parametros ou canonical)
- [ ] **Canonical tag** definida
- [ ] **Paginacao** com rel=next/prev ou infinite scroll com URL updates
- [ ] **FAQ section** no fundo da pagina (3-5 perguntas relevantes)
- [ ] **Links internos** para subcategorias e artigos de blog relacionados
- [ ] **Schema markup** (CollectionPage ou ItemList)
- [ ] **Mobile responsive** — grelha de produtos adaptavel
- [ ] **Velocidade de carregamento** < 3 segundos

## 3. Checklist SEO para Paginas de Produto

### Elementos On-Page (OBRIGATORIO em cada produto)

- [ ] **Title tag**: "[Nome Produto] - Comprar [Marca] [Modelo] | VoltStock" (60 chars max)
- [ ] **Meta description**: Resumo de specs + preco + CTA (155 chars max)
- [ ] **H1**: Nome completo do produto
- [ ] **URL limpa**: voltstock.pt/[categoria]/[nome-produto-slug]
- [ ] **Imagens**: minimo 3, com alt text descritivo, formato WebP, tamanho < 200KB
- [ ] **Descricao unica**: minimo 200 palavras (NAO copiar do fabricante)
- [ ] **Especificacoes tecnicas**: tabela estruturada e legivel
- [ ] **Preco visivel** com schema markup (Product + Offer)
- [ ] **Disponibilidade** indicada (em stock / sob encomenda)
- [ ] **Breadcrumbs** com schema
- [ ] **Reviews/Avaliacoes** (mesmo que internas) com schema AggregateRating
- [ ] **Produtos relacionados** (cross-sell / upsell)
- [ ] **FAQ do produto** (2-3 perguntas especificas)
- [ ] **Botao de compra** acima da dobra (above the fold)
- [ ] **Canonical tag** para evitar duplicacao
- [ ] **Open Graph tags** para partilha social
- [ ] **Velocidade**: imagens lazy-loaded, pagina < 2.5s LCP

## 5. Estrategia de Linking Interno

### Principios

1. **Estrutura em silo**: Cada categoria principal tem subcategorias e artigos de blog que linkam entre si
2. **Link de produto para produto**: "Tambem te pode interessar" + "Frequentemente comprados juntos"
3. **Link de blog para produto**: Todo o artigo de blog linka para 3-5 produtos relevantes
4. **Link de produto para blog**: "Guia de compra" e "Como usar" na pagina de produto
5. **Breadcrumbs consistentes**: Home > Categoria > Subcategoria > Produto

### Mapa de Links Internos

```
Homepage
├── /placas-graficas (Categoria)
│   ├── /placas-graficas/nvidia (Subcategoria)
│   │   ├── /placas-graficas/nvidia/rtx-4070 (Produto)
│   │   └── /placas-graficas/nvidia/rtx-4060 (Produto)
│   ├── /placas-graficas/amd (Subcategoria)
│   └── Blog: "Melhor placa grafica 2026" ↔ Linka para categoria + 5 produtos
│
├── /perifericos (Categoria)
│   ├── /perifericos/teclados (Subcategoria)
│   │   └── Blog: "Melhor teclado mecanico" ↔ Linka para 5 teclados
│   ├── /perifericos/ratos (Subcategoria)
│   │   └── Blog: "Melhores ratos gaming" ↔ Linka para 5 ratos
│   └── /perifericos/headsets (Subcategoria)
│
├── /monitores (Categoria)
│   └── Blog: "Melhor monitor gaming" ↔ Linka para 5 monitores
│
├── /blog (Hub de conteudo)
│   ├── Guias de compra → Linkam para categorias + produtos
│   ├── Tutoriais → Linkam para produtos necessarios
│   └── Comparacoes → Linkam para os 2+ produtos comparados
│
└── /sobre-nos, /faq, /contacto (Paginas institucionais)
```

### Regras de Anchor Text

| Tipo | Exemplo | Quando usar |
|------|---------|-------------|
| Keyword exacta | "melhor placa grafica" | 20% dos links (com moderacao) |
| Keyword parcial | "as melhores placas graficas de 2026" | 30% dos links |
| Branded | "na VoltStock" | 20% dos links |
| Natural/Generico | "ver mais aqui", "descobre" | 20% dos links |
| URL pura | "voltstock.pt/placas-graficas" | 10% dos links |

### Accoes Prioritarias

1. **Cada artigo de blog** deve ter minimo 5 links internos para produtos/categorias
2. **Cada pagina de produto** deve ter 3-5 links para produtos relacionados
3. **Cada pagina de categoria** deve linkar para o guia de compra correspondente
4. **Homepage** deve linkar directamente para as 5 categorias mais importantes
5. **Footer** com links para todas as categorias e paginas institucionais

## 7. Checklist Tecnico de SEO

### Fundamentals

- [ ] **SSL** activo (HTTPS em todo o site)
- [ ] **Sitemap XML** gerado e submetido ao Google Search Console
- [ ] **Robots.txt** configurado (bloquear /admin, /api, /checkout)
- [ ] **Canonical tags** em todas as paginas
- [ ] **Hreflang** (se houver versao PT-BR no futuro)
- [ ] **Mobile responsive** (Core Web Vitals: LCP < 2.5s, FID < 100ms, CLS < 0.1)
- [ ] **Schema markup** implementado (Organization, Product, BreadcrumbList, FAQ, AggregateRating)
- [ ] **404 personalizada** com busca e links uteis
- [ ] **Redirects 301** para URLs alteradas
- [ ] **Velocidade** optimizada (imagens WebP, code splitting, lazy loading)

### Monitoramento

- [ ] **Google Search Console** verificado e monitorado semanalmente
- [ ] **Google Analytics 4** com eventos de e-commerce configurados
- [ ] **Ahrefs/Semrush** (ou alternativa gratis: Ubersuggest) para tracking de keywords
- [ ] **PageSpeed Insights** verificado mensalmente
- [ ] **Screaming Frog** audit trimestral (links partidos, redirects, meta tags)
